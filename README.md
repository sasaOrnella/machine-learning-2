# Machine Learning Portfolio

This repository contains machine learning and data analysis projects, covering marketing effect evaluation, natural language processing, and computer vision.


## Projects

### 1. Bayesian Analysis of Temu Super Bowl Advertising Impact
**Folder:** `ad-effect`

This project evaluates whether Temu's 2023 Super Bowl advertising campaign increased brand search interest in the U.S. market. Google Trends data was used as a proxy for consumer attention, and a Bayesian Interrupted Time Series model was built to estimate the post-campaign advertising effect.

**Tools:** Python, PyMC, ArviZ, pandas, Google Trends

---

### 2. Web Scraping and NLP Text Classification
**Folder:** `nlp-text-classification`

This project demonstrates an end-to-end NLP workflow, including static web text collection, text preprocessing, feature extraction, and text classification. The project includes web scraping examples using Japanese financial news and official announcement websites, and text classification experiments using public datasets such as AG News and IMDb.

**Tools:** Python, BeautifulSoup, scikit-learn, PyTorch, NLTK, pandas

---

### 3. Raccoon Object Detection with R-CNN
**Folder:** `object-detection`

This project implements an object detection pipeline for raccoon localization using VOC/XML annotations. The project follows an R-CNN style workflow, including region proposal generation, IoU-based sample assignment, classification, bounding box regression, and detection result visualization.

**Tools:** Python, PyTorch, TorchVision, OpenCV, NumPy, Matplotlib
