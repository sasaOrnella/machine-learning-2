# Bayesian Analysis of Temu Super Bowl Advertising Impact

This project evaluates whether Temu's 2023 Super Bowl advertising campaign increased brand search interest in the U.S. market. Google Trends data was used as a proxy for consumer attention, and a Bayesian Interrupted Time Series model was built to estimate the post-campaign advertising effect.

The purpose of this project is to provide a simple framework for evaluating the effectiveness of marketing actions such as advertising campaigns. In this project, Google Trends is used as a measurable indicator of brand attention due to data availability. In a real business setting, the same modeling logic could be applied to internal company metrics such as revenue, profit, website traffic, conversion rate, or customer acquisition. 


### Methods
- Intervention variable construction
- Bayesian Interrupted Time Series model
- Prior predictive check
- Posterior analysis
- Posterior predictive check
